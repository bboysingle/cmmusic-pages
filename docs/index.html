隐私策略
最后更新日期：2026年

概述

CMMusic（以下简称"本应用"）是一款开源音乐播放器。我们非常重视用户的隐私保护。本隐私策略说明了我们如何收集、使用和保护您的信息。

信息收集

我们不收集的信息

本应用不会：

  • 收集您的个人身份信息
  • 收集您的音乐文件或媒体内容
  • 跟踪您的使用行为
  • 向第三方分享您的数据
  • 使用分析或追踪服务

本地存储的数据

本应用会在您的设备本地存储以下数据：

  • 媒体源配置：您添加的 WebDAV、Samba、FTP、SFTP 等媒体源的连接信息（仅存储在本地）
  • 播放列表：您创建的播放列表信息
  • 播放历史：您的播放历史记录
  • 应用设置：您的应用偏好设置

⚠️ 这些数据仅存储在您的设备本地，不会上传到任何服务器。

网络连接

本应用需要网络连接以实现以下功能：

  • 远程媒体源访问：连接到您配置的 WebDAV、Samba、FTP、SFTP 等服务器
  • 网络电台：播放网络电台流媒体
  • 元数据获取：从音乐文件或者目录中获取专辑封面和歌词

这些连接仅用于访问您明确配置的媒体源，不会连接到我们的服务器。

权限说明

本应用请求的权限及其用途：

Windows 权限

  • 网络访问：用于连接远程媒体源和网络电台
  • 文件系统访问：用于访问本地音乐文件
  • 音乐库访问：用于管理和播放本地音乐库

数据安全

  • 所有数据均存储在您的设备本地
  • 媒体源密码使用 AES-256 加密存储
  • 不会向任何第三方传输您的数据
  • 代码完全开源，可供审查

第三方服务

本应用可能连接到以下类型的第三方服务：

  • 您配置的媒体源服务器：WebDAV、Samba、FTP、SFTP 等
  • 公开的音乐元数据服务：用于获取专辑封面和歌词
  • 网络电台服务器：用于播放网络电台

这些服务的隐私政策由其运营者制定，建议您查阅相关服务的隐私政策。

开源

本应用是开源软件，源代码完全公开。您可以：

  • 审查代码以验证隐私保护措施
  • 自行构建应用
  • 提交改进建议

项目地址：https://github.com/bboysingle/dioxusmusic

儿童隐私

本应用不针对 13 岁以下儿童。我们不会故意收集儿童的个人信息。

隐私策略变更

我们可能会不时更新本隐私策略。任何变更都会通过应用更新或项目仓库公告通知。

本应用遵守适用的法律法规。使用本应用时，请确保您有权访问所连接的媒体源和内容。
